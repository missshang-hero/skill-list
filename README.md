# My Skills

Personal agent skills for extending AI assistant capabilities.

## Available Skills

### chinese-translator

Automatically translate Chinese messages to English, respond in English, and optionally translate the response back to Chinese.

**Features:**
- Detect Chinese characters in user messages
- Translate Chinese to English automatically
- Process and respond in English
- Optional: Translate response back to Chinese

**Installation:**
```bash
npx skills add missshang-hero/my-skills@chinese-translator
```

### hot-stocks

获取A股市场热门股票数据，包括涨幅排行、涨停股票、板块热点等实时行情信息。

**Features:**
- 获取涨幅排行股票
- 获取涨停股票列表
- 分析热门板块
- 查询个股实时行情
- 大盘指数数据

**Installation:**
```bash
npx skills add missshang-hero/my-skills@hot-stocks
```

## Installation

Install specific skill:
```bash
npx skills add missshang-hero/my-skills@<skill-name>
```
